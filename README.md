# Основной код для тг, био, бота.

## Установка на LINUX:

Делаем установщик исполняемым:  
    `chmod +x install.sh`  
Запускаем установщик:  
    `./install.sh`  
Если всё успешно установлено:  
    `chmod +x bot.sh`  
Запуск:  
    `./bot.sh`  
  
## Установка на WINDOWS:

Устанавливаем пакеты:  
`pip install -r requirements.txt`  
Запуск:  
`bot.cmd`
